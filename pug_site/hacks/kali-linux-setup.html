What is Kali Linux?

Kali Linux is a Debian-based Linux distribution designed for advanced penetration testing and security auditing. It comes pre-installed with over 600 security and penetration testing tools.

Installation Methods

Method 1: VirtualBox Installation (Recommended for Beginners)

Step 1: Download VirtualBox

  1. Visit https://www.virtualbox.org
  2. Download VirtualBox for your operating system (Windows, macOS, or Linux)
  3. Install VirtualBox following the installation wizard

Step 2: Download Kali Linux

  1. Visit https://www.kali.org/get-kali/
  2. Download the "Pre-built Virtual Machines" version for VirtualBox
  3. Choose the 64-bit version (recommended)
  4. Extract the downloaded .7z file (you may need 7-Zip)

Step 3: Import Kali Linux into VirtualBox

  1. Open VirtualBox
  2. Click "File" ‚Üí "Import Appliance"
  3. Browse to the extracted .vbox file
  4. Click "Next" and then "Import"
  5. Wait for the import to complete (may take several minutes)

Step 4: Configure Virtual Machine Settings

  1. Select your Kali VM and click "Settings"
  2. System: Allocate at least 2GB RAM (4GB recommended)
  3. Processor: Allocate at least 2 CPU cores
  4. Display: Enable 3D Acceleration and set Video Memory to 128MB
  5. Network: Set to "NAT" or "Bridged Adapter"
  6. Click "OK" to save settings

Step 5: Start Kali Linux

  1. Select your Kali VM and click "Start"
  2. Wait for Kali to boot up
  3. Default credentials:
    Username: kali
    Password: kali
Change the default password immediately after first login using: passwd

Method 2: USB Bootable Installation

Step 1: Download Kali Linux ISO

  1. Visit https://www.kali.org/get-kali/
  2. Download "Installer Images" ‚Üí Choose your architecture (usually 64-bit)
  3. Download the ISO file (around 3-4 GB)

Step 2: Create Bootable USB

Using Rufus (Windows):

  1. Download Rufus from https://rufus.ie
  2. Insert a USB drive (minimum 8GB)
  3. Open Rufus and select your USB device
  4. Click "SELECT" and choose the Kali Linux ISO
  5. Keep default settings and click "START"
  6. Wait for the process to complete

Using Balena Etcher (Windows/Mac/Linux):

  1. Download Etcher from https://www.balena.io/etcher/
  2. Insert USB drive (minimum 8GB)
  3. Open Etcher, select the Kali ISO, select your USB drive
  4. Click "Flash!" and wait for completion

Step 3: Boot from USB and Install

  1. Insert the bootable USB into your computer
  2. Restart and enter BIOS/UEFI (usually F2, F12, Del, or Esc key)
  3. Change boot order to boot from USB first
  4. Save and exit BIOS
  5. Select "Graphical Install" from the Kali boot menu
  6. Follow the installation wizard:
    • Select language, location, and keyboard
    • Configure network (can skip if no internet)
    • Set hostname and domain
    • Create user account and password
    • Partition disk (use guided partitioning for beginners)
    • Wait for installation to complete

Post-Installation Setup

1. Update System

sudo apt update && sudo apt upgrade -y

2. Update Kali Tools

sudo apt dist-upgrade -y

3. Install Additional Tools (Optional)

sudo apt install kali-linux-everything

Note: This installs ALL Kali tools and requires significant disk space (15+ GB)

4. Configure Network Settings

sudo nano /etc/network/interfaces

5. Install Guest Additions (VirtualBox only)

  1. In VirtualBox menu: Devices ‚Üí Insert Guest Additions CD image
  2. Open terminal and run:
sudo apt install -y virtualbox-guest-x11
sudo reboot

Essential First Commands

Check System Information

uname -a

Check Network Configuration

ip addr show
ifconfig

Start/Stop Network Services

sudo systemctl start networking
sudo systemctl stop networking

Check Available Tools

ls /usr/share/kali-menu/

Recommended Settings

Enable SSH for remote access (use with caution):
sudo systemctl start ssh
sudo systemctl enable ssh
Create snapshots regularly in VirtualBox before major changes or tool installations
Use a separate network segment or isolated network for security testing
